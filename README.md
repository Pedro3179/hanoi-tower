# Tower of Hanoi Algorithm

This project solves the mathematical puzzle known as the *Tower of Hanoi*.
The algorithm takes a number as input and generates a list sorted in descending order.
The goal is then to move these numbers to the rightmost list (or rod).

This is part of the *certification project* and one of the requirements for earning the *Python Certification*.
The lab project is called "The Tower of Hanoi Algorithm."
